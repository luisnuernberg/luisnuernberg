## AI Development Workflow

I use AI as a development accelerator, not as a replacement for understanding the product.

My workflow usually includes:

- Breaking business problems into technical tasks
- Designing the structure and user flow before coding
- Using AI to generate, refactor and review code
- Testing features manually and validating business rules
- Improving prompts, documentation and project organization
- Reviewing the generated code before applying changes
